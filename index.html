About This App

The Spontaneous Outing Planner is an intelligent agent designed to help you discover new experiences. It uses Google's Gemini API and Foursquare API to interpret your desired "vibe" and suggest a creative two-stop plan. The backend logic then finds the best real-world locations for your outing, ensuring a seamless and enjoyable adventure.
